## Rebuild Timeline (5 Weeks)

**Start date:** March 19, 2026

| Week (Dates) | Focus | Key Outcomes |
| --- | --- | --- |
| Week 1 (March 23-27, 2026) | Rapid prototypes of key features | Clickable UI prototypes for homepage, member portal dashboard, FAQ hub, and contact flow. Feature list locked and scope confirmed. |
| Week 2 (March 30-April 3, 2026) | Design system + frontend foundation | Final UI kit (colors, type, components) and responsive page skeletons wired to mock data. |
| Week 3 (April 6-10, 2026) | Core build: backend + integrations | Auth, member data endpoints, and content management hooks ready; UI connected to live APIs. |
| Week 4 (April 13-17, 2026) | Polish + QA | Accessibility pass, performance tuning, cross-device testing, and content review completed. |
| Week 5 (April 20-24, 2026) | Final package + stakeholder outreach | Final build packaged, demo script prepared, and outreach to the Deputy Sheriffs' Association scheduled with presentation deck. |

## Work Split (3 Members)

- **Moiz (Backend):** API design, authentication, data models, integration with member data, deployment readiness.
- **Neil (UI):** Visual system, UI prototypes, responsive layouts, and final frontend polish.
- **Akhil (All-rounder):** Full-stack support wherever needed, QA/testing, and stakeholder presentation materials.
